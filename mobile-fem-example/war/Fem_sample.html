Finite Element Method Simulation - Running in the Browser and Uses Device Orientation

by Markus Sprunck

This experiment shows an FEM simulation in the browser. The implementation is based on paper.js and Google Web Toolkit. GWT helps to translate the Java code for the FEM simulation into JavaScript, which is then executed in the browser. The experiment has been developed with GWT 2.5 and App Engine SDK 1.7.3. It runs with Chrome Browser under iOS and Android. You may get the source code on GitHub.
